# virtualized-table

## Introduction
大数据量虚拟化表格组件


## Usage

### Install
```
$ npm install virtualized-table
```

### Example
```javascript
    import {VTable} from "virtualized-table";

    <VTable
      columns={columns}
      dataSource={dataSource}
      fixedLeftColumnCount={2}
    />
```
